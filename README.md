## 🌟 About

> **Telegraph X** သည် [Telegra.ph](https://telegram.org/blog/telegraph) minimalist publishing tool အတွက် Android client ဖြစ်ပါသည်။
> Telegram team မှ ဖန်တီးထားသည့် Telegra.ph ကို Android ဖုန်းမှ အဆင်ပြေစွာ အသုံးပြုနိုင်ပါသည်။

</div>

## ⚡ Features

</div>

<table>
<tr>
<td align="center" width="50%">

### ✏️ Publishing
- **Create** — article အသစ် ရေးခြင်း
- **Edit** — article ပြင်ဆင်ခြင်း
- **Delete** — article ဖျက်ခြင်း
- **Drafts** — changes auto-saving

</td>
<td align="center" width="50%">

### 🔧 Tools
- **Rich Blocks** — text, quote, heading, list, image
- **Multi-Account** — account များစွာ support
- **Stats** — publication view statistics
- **Night Mode** — dark theme support

</td>
